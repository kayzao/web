Memory & Storage

RAM, or Random access Memory, is what the CPU uses to save data quickly. Desktop RAM can be categorized by the letters DDR, followed by a number, with that number representing the generation of that memory. The current, and most widely used, generation of RAM is DDR4. It supports speeds of up to 3600mhz. The faster the speed of the memory, the faster applications can run, if they requiere constant use of memory. Different CPUs support different amounts of RAM. Usually, you would want at least 4gb of RAM if you want to run Windows. If you want to play games, then it is recommended to have 8-16gb of RAM. For memory intensive workloads, you should have at least 32gb of RAM

On your motherboard, you should have at least 2 slots of RAM for peak performance. For more information on memory, click here for a guide on the specifics of memory

There is a difference between memory and storage. Memory is volatile, meaning that it resets if power is lost. Storage, on the other hand, is permanent, as long as the item holding the data is not physically destroyed. There are two primary types of storage: Hard drives and SSDs.

SSD stands for Solid State Drives. They are the faster, but more expensive, storage option of the two. There are several types of SSDs, but all are faster, and more reliable than hard drives. They function by electronically saving data.

Hard drives are much cheaper than SSDs, but are less reliable, and take up more space. This is because unlike SSDs, hard drives save data on discs, and physically locate data.

For a computer, it is recommended for convience to have both an SSD and a hard drive. This is so that data that is constantly being used can be saved to a SSD, such as the operating system, and data that is less frequently used can be saved to a hard drive as bulk storage.
