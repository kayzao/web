About the Project:

What is the python program called and what does it do?

The program, called Typewriter, takes in a text file containing a book from Project Gutenburg, as well as some user input, to print out randomly generated words for the user to type, with most punctuation removed. The program then calculates the time taken for the user to finish typing, which words were incorrect, and which characters were incorrect. With this information, it is able to tell the user their WPM (words per minute), number of characters typed, accuracy of characters typed, accuracy of each character, and which words were typed incorrectly.

How do you use the project?

Download the python file, and run it. Make sure that you already have the module matplotlib downloaded. When prompted to, type in what the program requests.

What are the files used in the program?

They are AnnualCS_project.py, and source1_clean.txt. AnnualCS_project.py is the python code that does everything, and source1_clean.txt is the source for the random words.

How does the program print randomly generated words?

The text file with the book has already been edited to not have the table of contents and the header and footer. First, the text file is split into words, and added into a list. Then, a random number is generated from 0 to the length of the list minus one. The random number picks a word in the list, and if the word doesn't contain numbers, the word is added to a separate string. Once the requested number of words is generated, the program runs through the list, removing all characters if they are not letters or dashes. Finally, the words are printed.

How does it calculate WPM and other related characteristics?

The time taken for the user to type the words is calculated by saving the time when the user is prompted to start typing, and subtracts that from the time at when the user presses enter.

WPM is calculated by dividing the number of characters inputed by the number of seconds it took to do so, before dividing the result by 4.7, since 4.7 characters is a rough estimate for the average word length in English.

To find out which words were typed incorrectly, both the user's input and the printed words are split into separate lists containing their words. The program then compares each word to its corresponding word in the user's input, and if they do not match, then it is marked as a "wrong word".

To find out whcih characters were mistakes and which ones were correct, Typewriter looks at each character of each word from the answer and from the output. It goes through all the letters of a word in the answer, or in the output, depending on which one is shorter. If one of these letters is incorrect, the program marks it as such. If not, the program marks it as correct. Next, if the word from the answer is too long, the program counts the number of extra letters, and marks the last letter of the word from the output wrong that many times. If the word from the answer is too short, then it marks each letter of the output word that hasn't been compared yet as a mistake.

To find the accuracy of each letter typed, Typewriter simply prints out the number of times a character was typed correctly divided by the total number of times that character was typed, and converts it to a percentage.

Are there any notable bugs with the program?

Rarely, two words will be separated by not one space, but two spaces. This occurs since all words are generated prior to the removal of all symbols that aren't characters from the english alphabet or a dash, meaning words that are only made up of such symbols are wiped, leaving behind some odd spacing. In addition, sometimes text will appear that will make it seem as though the program failed to separate two words by a space. In reality, those words were separated by a single period, comma, or other symbol. That symbol was then removed, leaving no separating symbol or space between the words. There is no solution to this other than changing the text file itself. If you encounter an instance of two words combined without a space separating them, type the words as they are, combined together.
