The Fundamentals of a Computer

This website discusses the fundamental components of a computer, specifically, a personal computer, or PC. Of a computer, there are several major parts. These are the:

  • Motherboard
  • Power Supply
  • Processor
  • Memory
  • Storage
  • Graphics source
  • Case

This website contains links to several other pages, each discussing one of the above components. Those links are located near the top of the page. In addition, clicking on the github logo in the top right corner of the screen will open the github source files for this website.
